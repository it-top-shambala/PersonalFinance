Классы

1. Categories - категории операций 
    Свойства: IdCategories - идентификатор  категории операции, Name - имя категории,
    Type - тип  категории (0 - расход, 1 - приход)

3. Currency - валюта
    Свойства: IdCurrency - идентификатор валюты, Name - имя валюты, Code - код валюты

4. Wallets - кошельки
    Свойства: IdWallet - идентификатор кошелька, Name - имя кошелька,
             IdCurrency - идентификатор валюты, Balance - сумма в кошельке
    Метод:  public bool WalletOperation(double summa) - проведение операции с кошельком,
    параметр summa - сумма операции
    
 5. Operations - операции
    Свойства: IdOperation - идентификатор операции,  IdWallet - идентификатор кошелька, DateTime - время проведения операции,
             IdCategory - идентификатор категории операции, Summa -  сумма операции
