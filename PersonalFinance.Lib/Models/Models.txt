Классы

1. Category - категори операции 
    Свойства: CategoryId - идентификатор  категории операции, Name - имя категории,
    Type - тип  категории (0 - расход, 1 - приход)

3. Currency - валюта
    Свойства: CurrencyId - идентификатор валюты, Name - имя валюты, Code - код валюты

4. Wallet - кошелек
    Свойства: WalletId - идентификатор кошелька, Name - имя кошелька,
              CurrencyName - имя валюты, Balance - сумма в кошельке
    Метод:  public bool WalletOperation(double summa) - проведение операции с кошельком,
    параметр summa - сумма операции
    
 5. Operation - операция
    Свойства: OperationId - идентификатор операции,  WalletId - идентификатор кошелька, DateTime - время проведения операции,
             CategoryName - имя категории операции, Summa -  сумма операции
