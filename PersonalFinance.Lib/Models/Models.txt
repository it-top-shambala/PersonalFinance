Классы

1. Category - категори операции 
    Свойства: Category_id - идентификатор  категории операции, Name - имя категории,
    Type - тип  категории (0 - расход, 1 - приход)

3. Currency - валюта
    Свойства: Currency_id - идентификатор валюты, Name - имя валюты, Code - код валюты

4. Wallet - кошелек
    Свойства: Wallet_id - идентификатор кошелька, Name - имя кошелька,
              Name_currency - имя валюты, Balance - сумма в кошельке
    Метод:  public bool WalletOperation(double summa) - проведение операции с кошельком,
    параметр summa - сумма операции
    
 5. Operation - операция
    Свойства: Operation_id - идентификатор операции,  Wallet_id - идентификатор кошелька, DateTime - время проведения операции,
             Name_category - имя категории операции, Summa -  сумма операции
