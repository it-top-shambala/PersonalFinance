Классы

1. Operation - Oперация (расход и приход)
    Свойства: Id - идентификатор операции, Name - имя операции, TypeOperation - тип  операции (0 - расход, 1 - приход)
3. Currency - валюта
    Свойства: Id - идентификатор валюты, Name - название валюты, Code - код валюты
4. Wallet - кошелек
    Свойства: Id - идентификатор кошелька, Name - имя кошелька,
             Currency - объект класса валюты, Balance - количество средств в кошельке
    Метод:  public bool WalletOperation(double sum) - проведение операции с кошельком, параметр sum - сумма операции
            положительное значение - сумма прихода, отрицательное значение - сумма расхода              
 5. LogOperation: Id - идентификатор записи операции,  IdWallet - идентификатор кошелька, DateTime - время проведения операции,
             IdOperation - идентификатор операции, Sum -  сумма операции
